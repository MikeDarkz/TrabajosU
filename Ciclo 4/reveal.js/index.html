Archivos de texto plano

Archivos de texto plano

Es un tipo de archivo de computadora que contiene solo caracteres de texto sin formato. En un archivo de texto sin formato, cada línea de texto es simplemente una secuencia de caracteres sin información de formato adicional, como tamaño de fuente, color o estilo.

Archivos XML

Archivos XML

Es un tipo de archivo que utiliza un lenguaje de marcado llamado Lenguaje de marcado extensible (XML) para almacenar y organizar datos en un formato estructurado.

Consisten en datos de texto encerrados dentro de una serie de etiquetas, similar al marcado HTML. Estas etiquetas definen la estructura y el contenido de los datos almacenados en el archivo, lo que facilita que las aplicaciones de software analicen y extraigan información del archivo.

Se usan comúnmente para almacenar e intercambiar datos entre diferentes aplicaciones y plataformas de software, ya que proporcionan un formato estandarizado que puede ser entendido fácilmente tanto por humanos como por máquinas. Por ejemplo, intercambiar datos entre un servidor web y un navegador web cliente.

Archivos JSON

Archivos JSON

Un archivo JSON es un tipo de archivo de computadora que utiliza un formato de intercambio de datos liviano llamado Notación de objetos JavaScript (JSON) para almacenar y transmitir datos en un formato estructurado.

Los archivos JSON consisten en datos de texto que se organizan en pares clave-valor, de forma similar a un diccionario o una tabla hash. Estos pares clave-valor están encerrados entre llaves {} y separados por comas, con cada clave y valor separados por dos puntos. Los archivos JSON se usan comúnmente para transmitir datos entre un servidor y una aplicación web, ya que proporcionan un formato estandarizado y liviano que puede ser fácilmente analizado tanto por humanos como por máquinas.


							{
								"name": "John",
								"age": 30,
								"city": "New York"
							}

Archivos Parquet

Archivos Parquet

Un archivo Parquet es un tipo de formato de almacenamiento en columnas que se utiliza para almacenar y procesar grandes cantidades de datos. A menudo se usa en marcos de análisis y procesamiento de big data, como Apache Hadoop y Apache Spark. Los archivos de parquet almacenan datos en un formato de columnas comprimido y altamente optimizado, lo que los hace eficientes tanto para leer como para escribir grandes cantidades de datos.

Una de las principales ventajas de los archivos de Parquet es su capacidad para procesarse en paralelo en un sistema distribuido, como un grupo de servidores o una plataforma de computación en la nube. Esto hace posible procesar grandes conjuntos de datos de manera mucho más rápida y eficiente que los formatos de almacenamiento tradicionales basados en filas. En general, los archivos de Parquet son una forma altamente eficiente y flexible de almacenar y procesar grandes cantidades de datos, lo que los convierte en una opción popular para una amplia gama de aplicaciones de análisis y procesamiento de big data.
